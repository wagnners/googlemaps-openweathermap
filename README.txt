-> To run an application, just open a file called "index.html" on the browser website. 
All libraries are already added to the project, as well as all the project's dependencies.

Obs: It wasn' possible to use "Open Weather" to calculate the weather forecast 1 hour interval. 
From what I've seen, the only API they provide for this is the "Hourly forecast" which is not available for the past "API Key". 
So, I used their API called "5 day / 3 hour forecast", which provides the forecast within 3 hours.
